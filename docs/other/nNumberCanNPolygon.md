# n个数值是否可以组成n边形

n个数值是否可以组成n边形。

## 示例

### 示例1

> 输入：[1, 1, 2]
>
> 输出：false

### 示例2

> 输入：[1, 2, 2]
>
> 输出：true

### 示例3

> 输入：[1, 1, 1, 4]
>
> 输出：false

### 示例4

> 输入：[1, 2, 3, 4]
>
> 输出：true
